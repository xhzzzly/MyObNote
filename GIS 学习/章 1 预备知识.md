# `Canvas`

## 绘制线段

```vue
<script setup>
import { onMounted, ref } from 'vue';

const canvasRef = ref(null);

onMounted(() => {
  const ctx = canvasRef.value.getContext('2d');
  ctx.beginPath();
  ctx.moveTo(100, 100);
  ctx.lineTo(200, 200);
  ctx.moveTo(200, 100);
  ctx.lineTo(300, 200);
  ctx.strokeStyle = 'blue';
  ctx.stroke();
  ctx.clostPath();
});
</script>

<template>
  <h1>This is a canvas</h1>
  <canvas ref="canvasRef" width="800" height="600" />
</template>

<style scoped></style>

```

## 绘制面

```vue
<script setup>
import { onMounted, ref } from 'vue';

const canvasRef = ref(null);

onMounted(() => {
  const ctx = canvasRef.value.getContext('2d');
  ctx.beginPath();
  ctx.moveTo(100, 100);
  ctx.lineTo(200, 200);
  ctx.lineTo(300, 100);
  ctx.fillStyle = 'blue';
  ctx.fill();
  ctx.clostPath();
});
</script>

<template>
  <h1>This is a canvas</h1>
  <canvas ref="canvasRef" width="800" height="600" />
</template>

<style scoped></style>

```
